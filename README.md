# DevOps Docker Terraform Project

## Overview

This project demonstrates a DevOps workflow using Docker, Terraform, and GitLab CI/CD to automate application deployment and infrastructure provisioning.

The goal of this project is to show how DevOps engineers automate infrastructure and container deployments using Infrastructure as Code and CI/CD pipelines.

---

## Technologies Used

* Docker
* Terraform
* GitLab CI/CD
* Linux
* Infrastructure as Code (IaC)
* DevOps Automation

---

## Project Structure

devops-docker-terraform-project

app/ — Application code
docker/ — Docker container configuration
terraform/ — Infrastructure as Code
.gitlab-ci.yml — CI/CD pipeline configuration

---

## CI/CD Workflow

1. Developer pushes code to GitLab
2. CI/CD pipeline is triggered
3. Docker builds the application container
4. Terraform provisions infrastructure
5. Application runs inside containerized environment
